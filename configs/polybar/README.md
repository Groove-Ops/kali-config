# Configuracion inicial Polybar#

El archivo 'config.ini' es el que controla la polybar, llama al archivo 'modules.ini' para añadir los modulos. Los mudulos son controlador por los scripts creados en la carpeta 'scripts'.

###

Se pueden añadir o quitar modulos si abrimos el 'config.ini', vamos casi abajo del todo y donde aparece 'modules right,center,left' se ponen o quitan los modulos deseados en el orden deseado

### 

La configuracion esta basada en un tema del entorno de TheGoodHacker, solo he tocado minimas cosas
Solo he creado el modulo para el Firewall, funciona con ufw
